# Eye Tracker Website

## Overview
The Eye Tracker Website is a project designed to showcase eye tracking technology and its applications. This website will serve as a platform for users to understand the functionalities of eye tracking, including data visualization through heatmaps, calibration processes, and machine learning analytics.

## Project Structure
```
Eye-Tracker-Website
├── src
│   ├── pages
│   │   ├── index.html          # Homepage of the website
│   │   ├── heatmap.html       # Placeholder for heatmap data
│   │   ├── calibration.html    # Placeholder for calibration process
│   │   └── ml-analytics.html   # Placeholder for machine learning analytics
│   ├── styles
│   │   └── style.css           # CSS styles for the website
│   ├── scripts
│   │   └── script.js           # JavaScript for interactivity
│   └── assets
│       └── images              # Directory for image assets
├── package.json                 # npm configuration file
└── README.md                    # Project documentation
```

## Setup Instructions
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Install the necessary dependencies using npm:
   ```
   npm install
   ```
4. Open the `src/pages/index.html` file in a web browser to view the homepage.
